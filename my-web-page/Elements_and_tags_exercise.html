Answers to questions for this exercise

What does a tag do? What does it mean?
    Two tags make up one element. An element tells the browser about information inside the opening and closing tags. For example, some elements are headers, paragraphs, strong emphasis, etc.
Can tags be used in conjunction with others?
    Yes, tags can be used in conjunction with others. For instance, if you want to emphasize a particular head or paragraph, you can use the strong element. The strong element will bold it so it stands out on the page.
Does a tag need attributes to work?
    Not necessarily. The strong tag can be used alone and still work, BUT it can also be used with the paragraph tag and work too.
